CLIENTE X SERVIDOR
No momento que você utiliza um serviço, você é o cliente. Exemplo: Navegador. Quando se clica em um link, você será redirecionado para determinado servidor, o qual retém os aqruivos/dados desejados, quando você os solicita, uma cópia é gerada e mostrada ao cliente.
JavaScript é uma tecnologia Cilente X Serviidor.

Conteúdo -----> Design ------> Interação
HTML ---------> CSS ---------> JS
Essas ferramentas compõem o Cliente.
